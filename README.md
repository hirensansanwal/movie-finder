# 🎬 Movie Finder App

## 📌 Description

This project is a web application that allows users to search for movies and view details such as title, year, and poster using a public API.

## 🔗 API Used

OMDb API (https://www.omdbapi.com/)

## 🎯 Objective

To build an interactive web application using JavaScript, API integration, and UI design.

## 🚀 Planned Features

* Search movies by name
* Display movie results dynamically
* Filter movies by year or type
* Sort movies alphabetically or by rating
* Responsive design for all devices
* Dark/Light mode (optional)

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)

## ▶️ How to Run

1. Clone the repository
2. Open index.html in a browser
